How to use the simulation:

Within the section "Input of the simulation" there should be an input- and checkbox for every variable in the json file. If the checkbox is checked and the simulation is updated, that variable will be changed to a range of values. This range works in the same way as the python range function, with the first value being the lower bound, 2nd value the upperbound and 3rd value the stepsize. These values can be configured in steps of 0.01.
Below these buttons is the "Update data" button.
This will save changes within the program to be used when the "Start" button is pressed. This button needs to be pressed to switch a variable from a single value to a range or back.
